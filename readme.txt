Доступ к Вашим постам (на утверждении) по ссылке для модераторов.

== Description ==

Иногда существует необходимость показать пост удаленному человеку перед публикацией. С этим плагином Вы можете давать доступ к постам со статусом "на утверждении" по ссылке.
При этом не важно открыта или закрыта регистрация на вашем сайте.
Доступ можно защитить ключем, для лучшей конфиденциальности. При этом ключ будет добавлен к ссылке.
Сам ключ храниться в произвольном поле (mi13-access-by-link-key) записи и автоматически удаляется при публикации.

1) Создайте пост со статусом "на утверждении".
2) Перейдите на страницу плагина (Настройки > mi13 access by link).
3) Настройте front end вывод на основе вашей темы (single.php и т.п.) и ваших функций форматирования (functions.php и др.)  либо оставьте всё как есть.
4) При необходимости добавьте ключ поставив флажок.
5) Сохраните изменения.
6) Скопируйте ссылку для поста и перешлите ее вашему доверенному лицу.


== Installation ==

1. Скопируйте папку плагина <strong>mi13-access-by-link</strong> в <strong>/wp-content/plugins/</strong>. 
2. Активируйте плагин через меню <strong>Плагины</strong>.
3. Это все.
